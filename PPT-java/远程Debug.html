我们将代码部署到服务器上之后,如果代码出现问题,一般情况下,我们通过打各种日志,查看各种日志,再进行修改, 是非常不方便的.这个时候我们可以通过远程连接对代码进行调试,找出问题所在.

Java远程调试原理
