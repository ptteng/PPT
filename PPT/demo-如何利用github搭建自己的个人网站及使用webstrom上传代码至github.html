1.背景介绍

github是个什么东西？

在github的网站首页说的很清楚了。

1.github是一个基于git的web 协作社区，它有多种机制让大家协同的和你一起的对项目进行贡献。

2.github全平台、全设备支持，你可以在任何终端上，随时随地的共享你的代码、评论或着其他有意义的信息。

3.这个世界上最大的开源软件社区。

2.知识剖析

使用github有什么好处？

1.有一个好的github，人们能更快速的了解你的技术，兴趣涉猎。

2.方便进行项目托管，代码管理

3.如果你技术厉害的话，能在上面扩充人脉（因为会有志同道合的人找你）。

4.在面试上加分，获得一份工作，特别是高尖端的技术人群。
